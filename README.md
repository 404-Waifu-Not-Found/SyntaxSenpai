An AI-powered dating app designed for developers, where users choose a personalized “waifu” companion that evolves into a fully functional computer agent.

Users begin by selecting their ideal waifu through an interactive, dating-style interface. Each waifu has a unique personality, tone, and communication style. Once matched, the waifu becomes an intelligent assistant that lives on your device (PC or mobile), capable of executing real tasks.

From writing code and managing files to automating workflows, your waifu doesn’t just chat—she acts. Every interaction is delivered in-character, blending emotional engagement with real productivity.

Quickstart

1. Requirements: Node >= 25, npm >= 11
2. Copy environment template and fill keys: `cp .env.example .env.local`
3. Install dependencies: `npm install`
4. Run desktop app: `npm run dev:desktop`
5. Run mobile app: `npm run dev:mobile`
6. Build: `npm run build`
7. Tests & typecheck: `npm run test && npm run typecheck`

See PROVIDERS.md and PROVIDER_SETUP.md for provider configuration and environment variables.
