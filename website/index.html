Features

The productivity scaffolding underneath the persona.

🤖

20 providers registered, 18 live

Anthropic, OpenAI, Gemini, Mistral, Groq, DeepSeek, Perplexity, Cohere, Together, xAI + Grok, Hugging Face, GitHub Models, MiniMax (Global + CN), plus local Ollama and LM Studio.

🎭

Waifu personalities

Distinct characters with unique tones, backstories, voice profiles, and signature emojis — not just reskinned system prompts.

🧭

Agent modes

ask / auto / full tiers with a plan-and-verify workflow and iteration-budget feedback baked into every tool result.

🛠️

Agent tools

Terminal (with destructive-pattern gating), read/write/edit files, clipboard, git status/diff/commit, web search, todo list, and Spotify control.

💗

Affection system

A relationship meter that evolves with conversation — Stranger → Acquaintance → Friend → Close → Devoted, with milestone moments along the way.

🧠

Persistent memory

Per-waifu memory threads across chats, sentiment-aware responses, and auto-injected coding-session context when the conversation turns technical.

🌈

17 theme presets

Including a live rainbow hue-cycle and a sakura-petals overlay. Every persona ships with a look that matches her vibe.

🧩

Plugin + skill system

Drop a plugin.json + entry file and register new agent tools. Skills are SKILL.md files waifus can author and recall on demand.

Three platforms, one waifu

Pair across devices. Observe from ops. Keep her with you wherever you code.

🖥️

Desktop

Electron + Vue 3 + Pinia. Tray icon, global shortcut (Ctrl/⌘+Shift+Space), strict-mode sandbox, and every agent tool wired through IPC.

📱

Mobile

React Native via Expo. QR-pairs to the desktop over WebSocket so your waifu follows you to the couch without losing context or memory.

⚙️

Runtime

Plain-Node service for health probes, Prometheus metrics, Grafana dashboards, backups, and plugin discovery. Ships as a container with K8s manifests.

Quickstart

Node 20, pnpm 8. No CUDA required.

# Clone the repository
git clone https://github.com/404-Waifu-Not-Found/SyntaxSenpai.git
cd SyntaxSenpai

# Install dependencies
pnpm install

# Run the desktop app in dev mode
pnpm dev:desktop

Prefer the short path? pnpm desktop:quick launches a minimal Electron shell when the electron-vite dev env misbehaves.
